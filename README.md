# Workflow Visualizer PCF

A custom Power Apps Component Framework control for model-driven apps that visualizes workflow stages as a horizontal progress/status bar.

## Goal

Replace or supplement standard Business Process Flow bars with a configurable visual workflow bar that reflects workflow engine stage/task status.

## Status colors

- Passed: green
- In progress: yellow
- Skipped: grey
- Denied: red
- Upcoming: neutral grey

## Clean-room notes

This component is designed from scratch for internal learning and enterprise use. It does not copy proprietary source code, UI structure, naming, CSS, or implementation details from any third-party product.
